# Layout & Theme

All web pages should use the same layout. It

The design of the web page is mobile-first. Main content should be centered and condensed with minimal space. It must support hand gesture.

The layout should have a cute style, that attracts kids and teenagers.


## Color

Should have two main colors: primary and secondary.

Should be purple or some dark color that can view in low light environment.


## Font family

Font family should be able to display Vietnamese language.

Font size should be responsive and large enough to view without zooming. It is largest in widescreen but not too big.


## Inputs & Controls

Button should be rounded, with a thick border that's easy to know that it's clickable. It may have animation on hover or clicked.

Textbox should be transparent with a border display on focus.
