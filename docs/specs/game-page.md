# Game page

The game page is where user can:
- Change their default name
- Configure game's settings
- Invite other players
- Start playing

## Changing player's name

There should be a text input that contains current player's name.

In the first time user visit the page, name should be auto generated.

Name is sync with browser's local storage and is auto filled in the text input when user visit the page.


## Configuring game setting

The available settings are different base on the selected game.

For Tic-Tac-Toe game, there's no setting.


## Inviting players

User can invite other player by clicking an **Invite** button. When clicked, they can:
- choose so send the link of the game to Messengers, or other applications
- copy the link to clipboard
- get the QR code of the game so they can share it with nearby players without knowing their contact

The invite feature should work on both mobile and PC, laptop.


## Playing game

There should be a standout **Start** button that allow user to start playing. Clicking on it will redirect user to the [Play page](./play-page.md) so all players can play together.
